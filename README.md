E-Commerce Tableau Dashboard

Project Overview

This project is an interactive Tableau dashboard built using an E-Commerce dataset.
It provides insights into Sales, Customer behavior, and Product performance.

Features
- Sales Dashboard (Monthly trend, Top products, Order priority)
- Customer Dashboard (Gender analysis, payment & device preference)
- Product Dashboard (Category-wise and month-wise analysis)
- Navigation buttons to switch between dashboards
- Interactive filtering across visuals

Tools Used
- Tableau Public
- Excel / CSV Dataset

Dashboard Preview
- Sales Dashboard
- Customer Dashboard
- Product Dashboard

Key Insights
- Identified top-performing products and categories
- Analyzed customer behavior and preferences
- Compared sales trends across months
